# Pizza-sales-dashboard

This project showcases an end-to-end data analysis workflow using SQL for data extraction & cleaning and Power BI for dashboard creation.

 Project Overview :

Connected Power BI to SQL database containing pizza sales transactions

Performed data cleaning, joins, and aggregations function using optimized SQL queries

Built a fully interactive dashboard to visualization:

Total Revenue & Orders & avg order value , total pizza sold , avg pizza per order using kpi

Sales and total pizza category by Pizza Category & Size

Top 5/Bottom 5 Selling Pizzas,revenue,quantity

Daily/Monthly Sales Trends

Customer Order Insights


Implemented DAX formulas for KPIs and advanced calculations

Designed intuitive visuals and applied professional dashboard formatting

Tools & Technologies use

Power BI (Data Modeling, DAX, Dashboard Design)

SQL (Queries, Joins, Aggregations, Data Cleaning)

Excel / CSV (Initial Data Source)
